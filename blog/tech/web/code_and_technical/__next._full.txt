## 1. Mathematical Formulations ($\LaTeX$ / KaTeX)

### Inline Equations
* Mass-energy equivalence: $E = mc^2$, where $c \approx 3 \times 10^8 \text{ m/s}$.
* Euler's identity: $e^{i\pi} + 1 = 0$.
* Quantum state representation: $|\psi\rangle = \alpha|0\rangle + \beta|1\rangle$ with probability norm $|\alpha|^2 + |\beta|^2 = 1$.

### Display Equations

#### Gaussian Integral & Limits
$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$

#### Continuous Fourier Transform
$$
f(x) = \int_{-\infty}^{\infty} \hat{f}(\xi) \, e^{2\pi i \xi x} \, d\xi
$$

#### Matrix Algebra & Linear Systems
$$
\begin{bmatrix} a & b \\ c & d \end{bmatrix} \begin{bmatrix} x \\ y \end{bmatrix} = \begin{bmatrix} ax + by \\ cx + dy \end{bmatrix}
$$

#### Quantum Entanglement State (GHZ State)
$$
\mathcal{L}_{\text{GHZ}} = \frac{1}{\sqrt{2}} \left( |000\rangle + |111\rangle \right)
$$

#### Piecewise Defined Function
$$
f(n) = \begin{cases} 
\frac{n}{2} & \text{if } n \equiv 0 \pmod{2} \\ 
3n + 1 & \text{if } n \equiv 1 \pmod{2} 
\end{cases}
$$

from typing import List

def calculate_fibonacci(n: int) -> List[int]:
    """Generates Fibonacci sequence up to n terms."""
    if n <= 0:
        return []
    sequence = [0, 1]
    while len(sequence) < n:
        sequence.append(sequence[-1] + sequence[-2])
    return sequence[:n]

if __name__ == "__main__":

### Rust (Concurrent Memory Safety & Traits Test)

```rust
use std::sync::{Arc, Mutex};
use std::thread;

pub trait Processable {
    fn process(&self) -> Result<(), String>;
}

struct Worker {
    id: usize,
}

impl Processable for Worker {
    fn process(&self) -> Result<(), String> {
        println!("Worker {} processing job.", self.id);
        Ok(())
    }
}

pub fn run_parallel_jobs(count: usize) {
    let counter = Arc::new(Mutex::new(0));
    let mut handles = vec![];

    for i in 0..count {
        let counter_clone = Arc::clone(&counter);
        let handle = thread::spawn(move || {
            let worker = Worker { id: i };
            let _ = worker.process();
            let mut num = counter_clone.lock().unwrap();
            *num += 1;
        });
        handles.push(handle);
    }

    for handle in handles {
        handle.join().unwrap();
    }
}
```
