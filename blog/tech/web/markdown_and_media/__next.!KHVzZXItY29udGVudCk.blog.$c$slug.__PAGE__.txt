## 1. Typography & Inline Elements

Standard Markdown formatting supports standard inline styling and HTML5 semantic inline markups:

* **Bold Text** for explicit emphasis (`**Bold Text**`).
* *Italic Text* for subtle emphasis (`*Italic Text*`).
* ***Bold and Italic*** combined (`***Bold and Italic***`).
* ~~Strikethrough~~ for deprecated content (`~~Strikethrough~~`).
* `Inline code` for variables and expressions (`const version = "14.2.0";`).
