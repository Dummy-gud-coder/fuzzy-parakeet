MDX Media & Layout Specification Test Suite

Welcome to the unified MDX media and formatting test suite. This document consolidates all media, interactive prompt components, typographical elements, layout edge cases, and parser verification tests without redundancies.

1. Typography & Inline Elements

Standard Markdown formatting supports standard inline styling and HTML5 semantic inline markups:

  • Bold Text for explicit emphasis (**Bold Text**).
  • Italic Text for subtle emphasis (*Italic Text*).
  • Bold and Italic combined (***Bold and Italic***).
  • Strikethrough for deprecated content (~~Strikethrough~~).
  • Inline code for variables and expressions (const version = "14.2.0";).
  • External link: Next.js Documentation.
  • Subscript & Superscript test: H2O and E = mc2 (or X10).
  • Highlighted text test: Highlighted background text.
